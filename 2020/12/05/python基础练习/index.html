# -------------------------------------------------------------------------------
# Name:         第5题：输出字符奇数位置的字符串
# Description:  
# Author:       M4XLMUM
# Date:         2020/12/5
# -------------------------------------------------------------------------------
# 题目描述：
# 给你一个字符串 a， 输出a中奇数位置字符构成的字符串（位置编号从1开始）。
# 例如：a=‘xyzwd’
# 则输出:xzd
# a = '01234567'
for i in range((len(a)+1)//2):
    print(a[2*i], end='')
