题目

判断一个整数是否是回文数。回文数是指正序（从左向右）和倒序（从右向左）读都是一样的整数。

示例 1:

输入: 121
输出: true
示例 2:

输入: -121
输出: false
解释: 从左向右读, 为 -121 。 从右向左读, 为 121- 。因此它不是一个回文数。
示例 3:

输入: 10
输出: false
解释: 从右向左读, 为 01 。因此它不是一个回文数。
进阶:

你能不将整数转为字符串来解决这个问题吗？

来源：力扣（LeetCode）
链接：https://leetcode-cn.com/problems/palindrome-number

思路

一.因为输入均为整数，所以如果是负数直接return false，个位数return true(减少时间，个位数也可以不直接return true，后面的len>=2也可以解决)，其它的就用%取出最后一位，/(10的数字长度-1的次方)取出最高位，比较即可。
二.上一题的整数反转，判断反转后与原数是否相等。

代码实现

class Solution {
public:
    bool isPalindrome(int x) {
        if(x<0)
        {
            return false;
        }
        else if(x<10)
        {
            return true;
        }
        else
        {
            int len=0;
            for(int n=x;n!=0;n/=10)
            {
                len++;
            }
            int p=1;
            for(int m=1;m<len;m++)
            {
                p*=10;
            }
            for(;len>=2;)                          //len>=2不需考虑奇偶。
            {
                if(x%10!=x/p)                      //依次比较数字的第一位和最后一位
                {
                    return false;
                }
                len-=2;
                x%=p;                              //比较完之后消去第一位和最后一位
                x/=10;
                p/=100;
                
            }
            return true;
        }

    }
};
