这个是个浏览器的插件，就不需要再说了，去谷歌应用商店上面搜索安装一下就行了。然后就是配置了，注册一下账号，把那个什么助记词一定要记下来，到时候忘了密码，如果你有有价值的虚拟币的话，那就gg。当然如果你像我一样只在钱包里面玩玩没价值的东西的话，就整个记得住的密码就行了。

合约编写环境

这里有两个选择

  1. 可以选择自己搭建一个Remix的本地环境，但是麻烦的一批，具体怎么搭建可以自己搜一下(不推荐自己搭建，因为真的很麻烦我感觉)。按这个网站上的教程安装到你的本地计算机。

  2. 另一个选择当然就是在线的啦，而且还有中文的IDE，非常滴银杏，中文IDE的Remix在想网站、非官方的英文在线网站、官方的英文在线网站。

上面的在线网站自己看哪个适合自己，当然你也可以百度一下看看有没有更适合的，毕竟我也是还没入门，也不太清楚怎么才好。

智能合约编写学习

hello world!

因为博客不支持solidity语言的代码高亮，故这里将solidity的代码块搞成javascript方便代码高亮。

pragma solidity ^0.5.1;  //使用合约的版本

contract SimpleStorage {  //一个简单的存储合约
    uint storedData;   //无符号整数声明

    function set(uint x) public {  //合约里面包含的函数
        storedData = x;
    }

    function get() public view returns (uint) {
        return storedData;
    }
}

将上面的内容复制到在线的solidity的ide中后编译就行了，记得也许需要选择对应版本的编译器。。。

然后点击进入右边一个的运行栏目，点击部署，在下面选择已记录的交易，点击保存，再点击运行应该就行了。

经测试，我的在线环境只有上面的是javascript的时候才能正常生成与部署合约

但是在这里的话似乎就不用那个插件钱包了。。。直接点击创建账号就能创建一个有100ether的虚拟账户。那就更方便了，直接就这样好了。

但是这个似乎并不能将合约部署到网络上，只能在使用的ide里面进行调试与运行。emmm，明天再研究研究。
